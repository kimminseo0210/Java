아스키 코드와 유니코드
====
## 1. 아스키코드
<b>아스키코드(American Standard Code for Information Interchange)</b>로 영문 알파벳과 키보드에 있는 문자들을 정리한 대표적인 문자 인코딩 방식  
- 문자를 컴퓨터가 이해할 수 있는 이진 데이터로 혹은 그 반대로 변화하는 표준방법  
- 아스키 코드는 0~127까지 총 128개의 문자를 나타낼 수 있음

![AsciiImage](https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https%3A%2F%2Fblog.kakaocdn.net%2Fdna%2FbtOMFu%2Fbtq6XPqLXIR%2FAAAAAAAAAAAAAAAAAAAAANgGb-3fk1_FKhiJjCDqnqc7_klIISMq-r9Rjz-ODWM9%2Fimg.png%3Fcredential%3DyqXZFxpELC7KVnFOS48ylbz2pIh7yKj8%26expires%3D1769871599%26allow_ip%3D%26allow_referer%3D%26signature%3DAQfh6t7mgJZ12wU9Qc1T1Mga5Ag%253D)

A를 8비트로 변환하면 01000001이 되고 이를 십진술로 바꾸면 2의0승 + 2의6승으로 65 -> 문자 A가 된다  
###### 하지만 영문 알파벳과 기본 문구들만 인코딩이 가능하다보니 ASCII코드에는 한계가 있음, 주로 영어를 기반으로 설계가 되어있어 다른 언어의 문자나 특수기호를 표현하기에 어려움이 있음

## 2. 유니코드
<b>16비트 32비트 문자체계, 모든 문자를 다루도록 설계된 표준 문자 전산 처리방식</b>  
전 세계의 모든 문자를 컴퓨터에서 일관되게 나타내고 처리하기 위한 국제적인 표준 문자 전산 처리법
- 유니코드는 특정 글자를 표기할 때 <b>"U+(4자리 or 16자리 숫자)"</b>로 표기
- A -> 65(10진수) -> U+0041(유니코드)
- "U+"는 코드 포인트를 나타내는 접두사이며, 65를 16진수로 변환하면 41이 되고, 이 경우 코드 포인트는 0041이 됨
- 코드 포인트 : 유니코드에서 각 문자나 문자열을 고유하게 식별하기 위한 번호/식별자를 의미

## 3. 코드 활용
- python : 별도의 형변환 없이 내장함수를 사용 -> 매우 직관적
    ```python
    char = 'A'
    # 문자->숫자 ord()
    print(ord(char)) # -> 65 
    # 숫자->문자 chr()
    print(chr(65))  # -> 'A'
    ```
- java : char타입이 내부적으로 숫자(유니/아스키)로 저장되기 때문에, <b>강제 형변환(Casting)</b>을 주로 사용
  ```java
    public class Class1 {
        public static void main(String[] args) {
            char ch = 'A';
            int ascii = (int)ch;    // int로 강제 형변환 
            System.out.println(ascii);  // -> 65
            int code = 65
            System.out.print((char)code);   // -> A
        } 
    }
    ```
- javaScript : 문자열 객체인 메서드 `charCodeAt()`과 `String`클래스 메서드 활용
  ```javascript
    const char = 'A';
    const asciiCode = char.charCodeAt(0);
    console.log(asciiCode); // -> 65
    const originalChar = String.fromCharCode(65);
    console.log(originalChar);  // -> A
    ```
