자바 사용설명서
===
## 1. `while`문 입력으로 정지
가장 권장되는 방식은 무한 루프 내에서 `if` 조건문과 `break`를 사용하는 것
```java
import java.util.*;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        while (true) {
            int a,b = sc.nextInt();
            if (a==0 && b==0) {
                break;
            }
        }
    } 
}
```
## 2. `charAt(i)` 함수 : 문자열에서 문자 낚기
자바의 `String`은 배열이 아니므로 `[i] 대신 매서드를 사용해야 함
- 기능 : 문자열의 특정 인덱스(`i`)에 위치한 <b>문자열(`char`)</b> 1개를 가져옴
- 비교 : 파이썬의 `string[i]`와 동일한 역할
- 특징 : 반환값이 `char`타입이며, 컴퓨터 내부적으로는 <b>ASCII 번호</b>로 저장

## 3. `char`를 `int`로 변환하는 방법 (중요)
낚아챈 문자를 숫자로 다룰 때는 목적에 따라 방법이 달라짐

| 방법     | 코드        | 결과(문자'`1`'기준) | 용도                              |
|--------|-----------|---------------|---------------------------------|
| 강제 형변환 | `(int)ch` | 49            | 문자의 <b>아스키 코드 번호</b>가 궁금할때      |
|아스키 연산| `ch -'0'`  | 1             | <b>실제 숫자 계산</b>을 할 때 (가장 많이 사용) |
|내장함수|`Character.getNumericValue(ch)`|1| 코드를 읽기 쉽게 만들고 싶을 때              |

- 왜 `ch -'0'`인가 ? : 문자 `'0'`의 아스키 번호는 48로 `'1'`(49)에서 `'0'`(48)을 뺀 값이 1이 남는 원리를 이용

## 4. Java vs Python 문자열 접근 비교
| 구분    | Java                | Python                |
|-------|---------------------|-----------------------|
| 접근 방식 | `s.charAt(i)`       | `s[i]`                |
| 타입 성격 | `char`(기본형, 숫자와 호환) | `str`(객체, 숫자와 더하기 불가) |
| 숫자변환  |`ch -'0'`| `int(s[i])`           |
