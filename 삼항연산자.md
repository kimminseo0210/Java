삼항 연산자(Ternary Operator)
===

삼항 연산자는 `ìf-else`문을 딱 한 줄로 줄여주는 아주 좋은 도구  
이름이 삼항인 이유는 연산에 필요한 요소가 <b>3개(조건, 참일 때 값, 거짓일 때 값)</b>이기 때문  
코테나 실무에서 코드를 간결하게 만들 때 자주 쓰이니 익혀두는게 좋음

### 1. 기본 문법 (Java, JavaScript)
자바와 자바스크립트 문법이 똑같음
<b>(조건식) ? 참일때값 : 거짓일때값</b>  
예시 코드
```java
int score = 85;
String result = (score >= 60) ? "합격" : "불합격";
//--> 출력 : 합격
```
### 2. `if-else`vs 삼항연산자 비교
똑같은 로직으로 작성한 코드 비교

|방식| 코드 형태                                              |  
|----|----------------------------------------------------|  
|`ìf-else`| `int max;`<br/>`if (a>b) max=a;`<br/>`else max=b;` |  
| 삼항연산자| `int max = (a>b) ? a : b;`                          |

- 차이점 : `if-else`는 문장이라서 그자체로 값을 가질 수 없지만, 삼항 연산자는 식이라서 계산된 결과값을 변수에 바로 대입할 수 있음

### 3. 주의점(남용 금지)
삼항 연산자가 좋다고 너무 보잡하게 꼬아 쓰면 오히려 독이 됨
1. 가독성 : 조건 안에 또 삼항연산자를 넣는 '중첩 삼항 연산자'는 가급적 피해라
2. 타입 일치 : `:`앞뒤의 값은 가급적 같은 데이터 타입이어야 함

### 4. 요약
- 한 줄로 끝내고 싶을 때 사용
- 값을 변수에 바로 담고 싶을 때 아주 유용
- 구조 : `조건 ? 참 : 거짓`(자바,JS), `참 if 조건 else 거짓`
